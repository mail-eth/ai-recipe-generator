# AI Recipe Generator

Smart recipe generator using Xiaomi MiMo based on available ingredients and dietary restrictions

## 🎯 Features

- Powered by Xiaomi MiMo API
- Simple and intuitive interface
- Real-time processing
- Token consumption: 200K tokens/day

## 🚀 Quick Start

```bash
git clone https://github.com/mail-eth/ai-recipe-generator.git
cd ai-recipe-generator
npm install
cp .env.example .env
# Add your MIMO_API_KEY
npm start
```

## 📊 Use Case

This project demonstrates practical AI integration using Xiaomi MiMo API for real-world applications.

## 🛠️ Tech Stack

- Xiaomi MiMo API
- Node.js
- Express
- Simple HTML/CSS frontend
